About

Ball Game is a physics-driven 3D game where the player controls a rolling ball through environments using realistic momentum and collision response. The project focuses on low-level engine work — physics simulation, camera behavior, and responsive controls — built primarily in C++ with shader support for rendering.


Technical highlights


  
— Custom camera system with smooth follow behavior and dynamic pivot control

  
— Physics-based movement using force/torque application for realistic rolling feel

  
— C++ core with C# scripting layer and GLSL/ShaderLab shader pipeline

  
— Cross-platform support via Objective-C++ bridge
